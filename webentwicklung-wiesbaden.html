Bundesweit aus Wiesbaden.

Webseiten programmieren lassen.

PSD zu HTML Service.

Für all diejenigen, die bereits ein Design besitzen, mit einem anderen Designer zusammenarbeiten oder selbst Benutzeroberflächen gestalten, biete ich ebenfalls einen seperaten Programmierservice an. Dazu gehört die Programmierung von Photoshop-Templates oder von Sketch bzw. Adobe Experience Design Vorlagen. Ich programmiere dein Design 1 zu 1 und schicke dir im Anschluss die fertigen Codes.

von HTML bis JavaScript

Mit meiner Webentwicklung, verbinde ich das Design mit den richtigen Funktionen. Mein Markup und meine Codes entstehen mit den neusten Versionen von HTML5, CSS3 in Verbindung mit SASS, Vanilla Javascript ES6 und serverseitigen Programmiersprachen wie PHP.

Alle meine Webseiten werden von Hand programmiert und ich arbeite nicht mit WYSIWYG-Software. Ich arbeite als Full-Stack Developer und source nichts aus. Die Gestaltung der Informationsarchitektur erfolgt immer mit Fokus auf den Nutzer. Nach jeder Entwicklung, führe ich einen User Experience und einen Cross-Browser Test durch.

So stelle ich sicher, dass alles wie geplant funktioniert. Ganz egal, ob deine Seite mit Safari, Firefox Quantum oder Chrome abgerufen wird. Ich halte mich außerdem, an alle Regeln des World Wide Web Consortiums (W3C) und besuche wichtige Webdesign-Konferenzen, um mich mit den neusten Standards und Trends zu versorgen.

von WordPress bis Jekyll

Jede Website bei der Inhalt verändert werden kann oder Funktionen beliebig hinzugefügt werden können, sollte mit einem Content-Management-System arbeiten. Ich nutze dafür WordPress. WP ermöglicht es sogar nicht technikaffinen Personen, Webseiten an Änderungen des persönlichen Umfelds anzupassen.

Damit das ganze auch so leicht bleibt, wie eine einfache App-Installation, passe ich WordPress haargenau auf deine Website an. Daneben biete ich für all diejenigen, die bereits eine WordPress Webseite besitzen, einen umfassenden WordPress Service an.

Für statische Websiten nutze ich Jekyll in Verbindung mit den GitHub Pages, mit diesem Zusammenspiel, hast du die Möglichkeit deine statische Seite kostenlos auf dem GitHub Server zu hosten.
